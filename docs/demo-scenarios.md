# Demo Scenarios

1. Low toner autonomous procurement
   - Trigger low toner event
   - Inventory Agent checks stock and forecasts demand
   - Workflow generates purchase order and audit trace

2. Predictive maintenance
   - Device reports high temperature and repeated faults
   - Maintenance Agent scores failure risk
   - Executive Advisor ranks business impact by SLA exposure

3. Contract intelligence
   - Contract Agent extracts uptime obligations
   - System compares telemetry uptime with SLA
   - Recommendation includes evidence and violation risk

4. Secure support answer
   - Support user asks for troubleshooting guidance
   - RAG blocks prompt injection, masks PII, cites maintenance source

5. Customer success intervention
   - Churn model detects rising risk
   - Customer Success Agent recommends outreach and upsell action
